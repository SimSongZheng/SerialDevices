# Serial Device Emulator

- Author: ssz
- Last Update: 2026/07/13

## Introduction

This program is for emulating serial port devices for testing purpose.

## Preliminary

Python3 is expected to run this program.

## How to Run the Source code

1. Open the directory in vscode or powershell.
1. Run `python -m venv venv` to create a new virtual environment.
1. Run `pip install -r requirements.txt` to install python packages.
1. Run `.\venv\Scripts\activate` to activate the virtual environment; a (venv) will show in your prompt.
